## ✏️ 사용법

1. **공모 정보**부터 채워 마감일·제출 규정을 확정
2. **워크플로**에서 각 단계 상태를 `대기 / 진행 / 완료`로 갱신
3. 제출 직전 **제출물 점검표**로 최종 확인

> 항목은 일반적인 공모 기준 예시입니다. 실제 지침서(RFP)에 맞게 수정하세요.
