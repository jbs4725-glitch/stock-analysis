# 🎒 짐 싸기 리스트

> 스위스는 고산지대 일교차가 큽니다. 여름에도 방한·우비 준비 권장.

## 👕 의류
- [ ] 방수 재킷 / 우비
- [ ] 경량 패딩 or 플리스 (산악지대용)
- [ ] 편한 트레킹화 / 운동화
- [ ] 양말·속옷 (일수 + 여유분)
- [ ] 모자·장갑·선글라스
- [ ] 잠옷

## 🔌 전자기기
- [ ] 휴대폰 + 충전기
- [ ] 보조배터리
- [ ] **스위스용 어댑터 (Type J)** ← 중요
- [ ] 카메라 / 메모리카드
- [ ] 멀티탭(여러 기기 충전 시)

## 💊 비상 / 위생
- [ ] 상비약 (소화제·진통제·밴드·개인약)
- [ ] 자외선 차단제
- [ ] 세면도구
- [ ] 멀미약 (산악열차·유람선)

## 📄 서류 (별도 가방에)
- [ ] 여권 + 사본
- [ ] 항공권·숙소 예약확인서
- [ ] 트래블 패스
- [ ] 여행자보험 증서
- [ ] 비상 연락처 메모

## 💰 기타
- [ ] 스위스 프랑 현금
- [ ] 신용카드
- [ ] 보온병 (스위스는 식수 좋음)
- [ ] 에코백 / 장바구니
